# Keyboard & Palette

fordb is built to be driven without leaving the keyboard.

## Command palette

Press **`Ctrl/Cmd-K`** to open the command palette — a fuzzy launcher for every action: open a connection, run/format/explain the current query, export results, jump to a table, toggle panels, and more. If you can do it in fordb, you can do it from the palette.

## Core shortcuts

| Shortcut     | Action                                  |
| ------------ | --------------------------------------- |
| `Ctrl/Cmd-K` | Open the command palette                |
| `Mod-Enter`  | Run the current query                   |
| `Esc`        | Cancel a running query / close a dialog |

The palette is the source of truth for what's bound — open it and type to discover actions and their shortcuts. A full printable reference is on the way.
